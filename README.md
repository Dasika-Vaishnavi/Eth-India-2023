## How to install and run locally?

``
$ npm install
$ npm start
```

## Features -

- User Feed
- List all user posts
- Sort by Date & Trending
- Create a Post
- Like a post
- Edit & Delete Post
- User Profile
- Add Avatar
- Add Bio
- Add Portfolio URL
- Follow/Unfollow
- Explore Feed
- Bookmark Post
- Authentication
  - Sign-up
  - Login
  - Logout
- Loading and Alerts

## Built with -

- React JS
- React Context API + useReducer
- React Router v6
- Vanilla CSS
- Material UI Icons
- mockBee backend
- React Toastify
